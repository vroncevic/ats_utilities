# ATS Utilities - Use Cases

This directory contains scripts that demonstrate the functionalities of the `ats_utilities` framework.

## Available Examples

### 1. Option Parser
Demonstration of using `ATSOptionParser` with the `fire` strategy and the standard `argparse` approach.

**Execution:**
```bash
source local_env.sh
python3 story_ats_option_parser.py --help
```
